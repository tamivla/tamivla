**Специализация:** Создание production-готовых систем с полным циклом: от проектирования архитектуры и backend-разработки до настройки AI-пайплайнов и отказоустойчивой инфраструктуры.

**Уникальное сочетание:** 25-летний фундамент в системном администрировании и глубинное понимание инфраструктуры как основа для современных решений в DevOps, MLOps и интеграции AI.

## Технологический стек и компетенции

*   **DevOps & Инфраструктура:** Docker, Docker Compose, LXC/Proxmox, Nginx, CI/CD (GitHub Actions), Systemd, мониторинг.
*   **Backend-разработка:** Node.js, Express, Python, FastAPI, REST API, аутентификация (bcrypt, сессии).
*   **Базы данных:** PostgreSQL, расширение pgvector для векторного поиска, оптимизация.
*   **AI/ML Интеграция:** Работа с LLM (Yandex GPT API, локальные модели через Ollama/LM Studio), промпт-инжиниринг, векторные эмбеддинги (Hugging Face TEI), создание OpenAI-совместимых интерфейсов.
*   **Системная интеграция:** n8n (автоматизация, парсинг), Telegram Bot API, вебхуки, event-driven архитектура.

## Ключевые проекты

### Freelance Finder AI
**Production-готовый агрегатор фриланс-заказов с двухуровневой AI-фильтрацией.**

Полноценный SaaS-сервис, который переводит фрилансеров от ручного поиска к автоматическому получению релевантных предложений. Отвечал за полный цикл: проектирование микросервисной архитектуры, backend-разработку (Node.js/Python), интеграцию AI-моделей (Yandex GPT, локальные LLM, векторный поиск) и развертывание отказоустойчивой инфраструктуры (Docker, CI/CD, Nginx).

*   **Рабочее демо:** [finderlance.com](https://ff.tarasenko.me)
*   **Технический кейс и архитектура:** [github.com/tamivla/freelance-finder-ai-showcase](https://github.com/tamivla/freelance-finder-ai-showcase)

### Tamivla AI Server
**Proof of Concept MLOps-сервера для управляемой AI-инфраструктуры.**

Экспериментальный проект по созданию OpenAI-совместимого сервера управления локальными языковыми и эмбеддинговыми моделями в среде Windows. Задача — разработать работающую альтернативу облачным API под полным контролем, когда стандартные решения были недоступны. Успешно реализованный функционал (динамический менеджер моделей, работа с Hugging Face) обеспечил работу зависимых систем и дал глубокий практический опыт в проектировании API для AI.

*   **Репозиторий и описание:** [github.com/tamivla/tamivla-ai-server](https://github.com/tamivla/tamivla-ai-server)

## Принципы работы

*   **Полный цикл (Full-Cycle Ownership):** Готовность вести проект от идеи и архитектуры до production-деплоя, мониторинга и поддержки.
*   **Production-ориентированность:** Код и инфраструктура проектируются с учетом развертывания, безопасности, масштабирования и отказоустойчивости.
*   **Документация как стандарт:** Каждая система сопровождается технической документацией, схемами и инструкциями.
*   **Прагматичный выбор технологий:** Инструменты выбираются под конкретную задачу, с учетом поддерживаемости и стоимости владения.
